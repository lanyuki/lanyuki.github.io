git命令/hexok/ssh连接

发表于 2025-01-15 分类于 基础 Valine：
  • 连接远程仓库并推送代码

# 初始化一个新的 Git 仓库

git init

# 克隆一个远程仓库到本地

git clone <repository_url>

# 查看远程仓库

git remote -v

# 添加远程仓库

git remote add origin <repository_url>

# 推送本地分支到远程仓库

git push origin <branch_name>

# 拉取远程仓库的更新

git pull origin <branch_name>

# 从远程仓库获取更新但不合并

git fetch origin

  • 分支操作

# 查看所有分支

git branch

# 创建新分支

git branch <branch_name>

# 切换分支

git checkout <branch_name>

# 创建并切换到新分支

git checkout -b <branch_name>

# 合并分支

git merge <branch_name>

# 删除分支

git branch -d <branch_name>

  • 基本操作

# 查看当前状态

git status

# 添加文件到暂存区

git add <file_name>

# 添加所有文件到暂存区

git add .

# 提交暂存区的文件到仓库

git commit -m “Commit message”

# 查看提交历史

git log

# 查看简化的提交历史

git log –oneline

  • 撤销操作

# 撤销工作区的修改

git checkout – <file_name>

# 撤销暂存区的修改

git reset HEAD <file_name>

# 撤销最近的提交

git reset –soft HEAD~1

# 撤销最近的提交并丢弃更改

git reset –hard HEAD~1

  • 服务器配置 git

# 设置用户名

git config –global user.name “Your Name”

# 设置用户邮箱

git config –global user.email “your.email@example.com“

# 查看配置

git config –list

  • Hexo命令

# 生成静态文件

hexo generate/hexo d

# 启动本地服务器

hexo server/hexo s

# 部署到配置的 Git 仓库

hexo d

  • #新建文章

  • hexo name”文章名字”

  • #删除文章

  • rm path

  • ssh连接

#生成密钥对

ssh-keygen -t rsa -b 4096 -C “your_email@example.com"

#查看公钥

cat ~/.ssh/id_rsa.pub

#测试链接

ssh -T git@github.com

成功则是Hi your_username! You’ve successfully authenticated, but GitHub does not provide shell access.

VIT-BART-T5-yolos原理简述

发表于 2025-01-15 分类于 AI-learning Valine：
Vit模型，把一个个小图片过一个线性投影层，原来的图片像素点拉直（比如是一个128\*20\*20的图像，变成一个128\*400），乘上那个线性投影层层（w）变成一个个向量(粉色的)（即原来的128\*400现在变成了一个128\*1的向量）。模仿nlp,在每个位置都加一个位置向量，紫色加粉色就是一个位置加图像，两个进行位加，这个就代表这个图片的整体信息。最前面加一个cls，代表一个图片，然后采用mlp进行识别。
